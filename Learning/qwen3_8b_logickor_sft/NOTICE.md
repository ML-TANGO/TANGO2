# Notice

This package contains a minimal, reproducible subset of the LogicKor evaluation workflow used for Qwen3-8B SFT experiments.

- Original LogicKor repository: https://github.com/instructkr/LogicKor
- The included files under `logickor_eval/` are provided for experiment reproducibility and are based on the local evaluation workflow used in this project.
- Large artifacts such as model weights, training data, generated outputs, and evaluated outputs are intentionally excluded from Git and should be distributed separately.
