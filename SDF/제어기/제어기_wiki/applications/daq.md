# DAQ (Beymons RED)

## 제품 코드 설명

- 제품군 코드: DAQ
- 세부 제품 코드
  - GW: Gateway
  - ND: Node
  - SN: 환경 센서

- DAQ-GW

- DAQ-SN-HT-xy-md02
  - 온습도 센서
- DAQ-SN-TEMP-thermister
  - 온도 센서 (Thermister)
- DAQ-SN-TEMP-pt100
  - 온도 센서 (PT-100)
- DAQ-SN-TEMP-ktype
  - 온도 센서 (K-Type)
- DAQ-SN-PRES-wxyz
  - 압력 센서
- DAQ-SN-CT-wxyz

- DAQ-SN-VIB-thread
  - 진동 센서 (Thread type)
- DAQ-SN-VIB-magnetic
  - 진동 센서 (Magnetic type)
