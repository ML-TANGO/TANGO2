# Beymons 구성요소 간 명령 전송 관련 공통사항

일반적인 데이터 전송이 아닌 별도로 정의된 명령을 주고받기 위한 프로토콜

명령은 크게 다음 범주에 따라 구분된다.

- 명령 형태: get / set
- 메시지 타입: request / response
- 명령 방향: upstream / downstream
